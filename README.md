# 違反勞動基準法公司查詢

查詢各縣市違反勞動基準法的公司，取代難用的 PDF 格式，直接在網頁上搜尋與篩選。

**線上版：** https://thwsby2009-blip.streamlit.app

---

## 功能

- **縣市篩選** — 選擇特定縣市，快速找出當地違法企業
- **違反法條篩選** — 按違反法條（第24條、第30條、第32條等）過濾
- **關鍵字搜尋** — 輸入公司名稱關鍵字，快速找到目標
- **裁罰金額顯示** — 一眼看出裁罰金額
- **響應式表格** — 手機、電腦皆可瀏覽

---

## 常見違反法條說明

| 法條 | 說明 |
|------|------|
| 第24條 | 延長工時未依規定加給工資 |
| 第30條 | 工作時間記錄未逐日記載 |
| 第30-1條 | 輪班制未給予充分休息 |
| 第32條 | 延長工時超過上限 |
| 第38條 | 特別休假未依法折算工資 |
| 第39條 | 員工請假規定未核給 |

---

## 技術

- **前端**：Streamlit（Python）
- **資料**：CSV 格式（可由政府 PDF 轉檔而來）
- **部署**：Streamlit Cloud

## 本地運行

```bash
pip install -r requirements.txt
streamlit run app.py
```
