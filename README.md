# 違反勞動基準法裁罰查詢

查詢各縣市違反勞動基準法的公司，直接搜尋、不用下載 PDF。共收錄超過 8 萬筆真實裁罰資料。

**線上版：** https://thwsby2009-blip.streamlit.app

---

## 功能

- **縣市篩選** — 選擇特定縣市，快速找出當地違法企業
- **違反法條篩選** — 按違反法條（第24條、第30條、第32條等）過濾
- **關鍵字搜尋** — 輸入公司名稱關鍵字，快速找到目標
- **裁罰金額顯示** — 一眼看出裁罰金額與違反事由
- **分頁瀏覽** — 8 萬筆資料分頁呈現
- **響應式設計** — 手機、電腦皆可瀏覽

## 常見違反法條說明

| 法條 | 說明 |
|------|------|
| 第22條 | 工資未全額直接給付 |
| 第24條 | 延長工時未依規定加給工資 |
| 第30條 | 工作時間記錄未逐日記載至分鐘 |
| 第30-1條 | 輪班制未給予充分休息 |
| 第32條 | 延長工時超過上限 |
| 第38條 | 特別休假未依法折算工資 |
| 第39條 | 員工請假規定未核給 |

## 資料來源

資料來自 [勞動部公告系統](https://announcement.mol.gov.tw/)，每週更新。本系統收錄完整裁罰紀錄，包括：

- 事業單位名稱（含負責人）
- 縣市/單位別
- 處分日期與公告日期
- 違反法規條款
- 法條詳細說明
- 裁罰金額

## 更新資料

### 方式一：本地更新腳本
```bash
python update_data.py
```
執行後會自動從勞動部下載最新裁罰資料，取代 `data/labor_violations.csv`。約需 30-60 秒。

### 方式二：手動更新
1. 前往 [勞動部公告系統](https://announcement.mol.gov.tw/)
2. 選擇「縣市」與「法規：勞動基準法」
3. 下載格式選 **CSV**，點下載
4. 將 CSV 檔案取代 `data/labor_violations.csv` 即完成更新
